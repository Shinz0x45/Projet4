# Projet 4 – Application de gestion de budget personnel

Application web permettant à un utilisateur de gérer son budget personnel en enregistrant ses revenus et ses dépenses, de les catégoriser et d’analyser l’évolution de son solde.

Ce projet est réalisé dans le cadre de la formation Développeur Web Full Stack.

## Objectifs
- Concevoir une interface front-end simple en HTML/CSS, puis évoluer vers Angular
- Développer une API backend en Python (Flask dans un premier temps, avec une possible évolution vers Django)
- Mettre en place une base de données relationnelle en SQL

## Fonctionnalités 
- Création et connexion d’un compte utilisateur
- Ajout de revenus et de dépenses
- Catégorisation des transactions
- Analyse du budget (solde, répartition des dépenses)
- Affichage des données dans une interface front-end

## Technologies utilisées
- Backend : Python (Flask / Django)
- Frontend : HTML, CSS (Angular prévu par la suite)
- Base de données : SQL
- Outils : Git, GitHub
